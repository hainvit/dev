# Eniroment

## Install redis on ubuntu
* To install redis on ubuntu, go to terminal and type the following commands
```
$ sudo apt-get update
$ sudo apt-get install redis-server
```

## Start redis
* Following commands
```
$ redis-server
```

## Check if redis is working
* Following commands
```
$ redis-cli
```
* This will open a redis
```
redis 127.0.0.1:6379
```
