## Pylint
* Install pylint
```
$ pip install pep8
$ pip install pylint
```
* Check conventions with pylint and reading results
```
$ pylint strings.py
```

## PIP
* PIP is a package manager for Python packages
* Check pip verssions 
```
$ pip --version
```
* Install pip packages
```
$ pip install [package-name]
```
* Uninstall pip packages
```
$ pip uninstall [package-name]
```
* Show package lists
```
$ pip list
```
